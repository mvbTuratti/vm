Modelo

Na primeira semana após a aprovação do nosso projeto, tentamos realizar as compras dos materiais nas lojas localmente pois já estávamos atrasados e não queríamos esperar a entrega, porém materiais essenciais nós não achamos disponíveis para a venda, então resolvemos pedir pela internet mesmo.

Decidimos fazer a estrutura do projeto em mdf, modelamos no autocad como seria a “caixa” do smart bartender, porém ela ainda não está pronta.

Componentes

A aquisição das bombas peristáuticas se provou mais complexo do que o previsto, visto que nenhuma das lojas próximas possuia os componentes necessários, fizemos aquisição online das bombas e outros componentes serão adquiridos dia 22/07 para testagem imbutida no Raspberry Pi.

Raspberry Pi

Aprendemos o básico do RaspBerrypi e bootamos ele. Estamos avaliando qual a melhor forma para o host do site, temos a alternativa de utilizar o Ngrok para fazer o host do site dentro do RBP, porém o azure é nossa primeira alternativa, hosteando o site na nuvem e o RBP conversando com a nuvem e executar os comando localmente. e estamos vendo também sobre usar o azure IOT para deixar um link estático para acessar o site, o problema dessa solução é que o azure IOT é pago. Ou usar outra ferramenta que se comunique utilizando somente o ssh, porém nesse cenário não teríamos tanta segurança como no azure

Quando o assunto é bebida pode contar com os universitários.
